Prototyping Carrier Board for the Daisy Seed™ 

https://electro-smith.com/products/daisy-seed

This PCB is designed to breakout all the PINs of the Daisy Seed along with additional components such as push buttons, LEDs, potentiometers, I2C, and SD card interfaces along with enough room for a typical 830 point breadboard.  
All 40 Daisy PINs are available on both sides of the board with those on the left being mirrored/inverted for additional flexibility.

Notes:
  1. Push buttons are tied on one side to GND to be used with an input pullup GPIO.
  2. Potentiomenters are prewired on on the CW & CCW sides to +3V3 and GND.
  3. Audio In/Out jacks are directly connected to the Seed's audio in/out pins.
  4. If the mono audio in jacks are used left audio in is normalized to right audio in so that they will both be available for processing by the seed if nothing is connected to right audio in.
  5. JST connectors are provided to breakout the Audio In/Out and MIDI In/Out signals to external jacks (1/4" Phono, 5-pin DIN, etc.).
  6. Jumpers are provided to change the configuration between TRS MIDI-A and TRS MIDI-B.  It is pre-configured for MIDI-A.  If MIDI-B is desired then the jumpers for MIDI-A need to be cut and those for MIDI-B bridged.
